# Student-Feedback-System
Description:
A Java-based Student Feedback System that allows students to submit feedback on courses and faculty, and enables administrators to view and analyze feedback reports.

Features:

Students can submit feedback for multiple courses and teachers.

Admin can view overall feedback statistics and generate reports.

Ensures secure and structured storage of feedback data.

User-friendly interface for easy interaction.
